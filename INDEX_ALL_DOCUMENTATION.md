## ✅ WHICH FILES TO INCLUDE IN GITHUB

**INCLUDE these in GitHub (push with git):**
- All documentation files (*.md)
- All source code (src/ folder)
- All tests (tests/ folder)
- Configuration files (package.json, .env, etc.)
- Docker files (Dockerfile, docker-compose.yml)

**DON'T INCLUDE (via .gitignore):**
- node_modules/ folder
- Database files (.db)
- Cache files
- Logs
